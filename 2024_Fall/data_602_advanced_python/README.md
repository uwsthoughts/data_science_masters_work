Work produced as part of DATA 602 - Advanced Programming with Python
 
CUNY course description:
"In this course students will learn aspects of contemporary programming that are important for data gathering and analysis, including real-time programming, GUI design, interactive database programming, service-oriented architecture, data collection with and without databases, machine learning, data mining techniques, and GIS programming. Computer security issues will also be addressed, as will overall computer architecture. Students will be required to create a working system for a large volume of data using publicly available data sets."
